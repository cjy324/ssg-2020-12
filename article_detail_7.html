제목 : 5강. 비교연산자
JAVA에서 두 수의 크기를 비교하고 싶을 때에는 **비교연산자**를 사용할 수 있다. 비교연산자의 종류는 다음과 같다. "==", "<", "<=", ">", ">=", "!=" 수학에서 쓰이는 기호와 같다고 생각하면 된다.
단, 사용시 주의할 점은 " = " 기호를 쓸 때 일반 수학에서는 하나만 입력해도 되지만 JAVA에서는 " == " 처럼 2개를 입력해야 한다. 또한 " >= ", " <= " 기호를 입력할 때에는 반드시 " > ", " < " 기호가 " = " 기호 앞에 위치해야 한다는 점을 명심하자.
### 예시 ```java int a = 20; int b = 20; // a와 b가 같은지 비교하는 연산자 System.out.println(a == b); // 결과: true // a가 b보다 작은지 비교하는 연산자 System.out.println(a < b); // 결과: false // a가 b보다 작거나 같은지 비교하는 연산자 System.out.println(a <= b); // 결과: true // a가 b보다 큰지 비교하는 연산자 System.out.println(a > b); // 결과: false // a가 b보다 크거나 같은지 비교하는 연산자 System.out.println(a >= b); // 결과: true // a와 b가 다른지 비교하는 연산자 System.out.println(a != b); // 결과: false ```
